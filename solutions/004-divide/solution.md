# Solution: Divide

Implement a `divide` method that accepts positional or named parameters and returns the division result.

Key points:
- Validate parameter shape (array with two elements or object with `dividend`/`divisor`).
- Validate numeric types (integers or floats).
- Return a meaningful `Invalid params` (`-32602`) error for missing/invalid inputs or division by zero.
- Support notifications (no response) and batch semantics.

The repository contains example implementations in `solutions/004-divide/code/` (Perl and Python).
